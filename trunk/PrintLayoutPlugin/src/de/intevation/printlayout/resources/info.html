PrintLayoutPlugin - Help

pan tool

<left mouse bottun>
pans the sheet
<shift> + <right mouse button>
scales the sheet
<ctrl> + <right mouse button>
rotates the sheet

select tool

<left mouse button>
selects a object or changes between scaling and rotating mode
<shift> + <left mouse button>
deselects a object
<ctrl> + <left mouse button>
selects another object
